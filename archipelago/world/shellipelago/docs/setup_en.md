# Shellipelago Setup Guide

## Required Software

- Archipelago
- A modern web browser
- The Shellipelago browser game: [itch.io](https://dragonsidestudios.itch.io/shellipelago)

## Creating a YAML

Use the [player options page for this game](../player-options) to configure and export a Shellipelago YAML.

Adding destructible tiles or enemies to locations adds many checks and can significantly slow down a playthrough.

## Joining a MultiWorld

1. Generate or receive a room containing a Shellipelago slot.
2. Open the Shellipelago browser game.
3. Choose Play Online.
4. Enter the room host, port, slot name, and password.
5. Press Connect.

Items received while connected are applied by the browser client.

## Same Slot Co-op

Multiple people can connect to the same Shellipelago slot and play together. Progress is stored by Archipelago, so no local browser save is required to keep the slot in sync.
