# Shellipelago Setup Guide

1. Install `shellipelago.apworld` into Archipelago's `custom_worlds` folder.
2. Restart Archipelago.
3. Generate a player options template, or use the included `Shellipelago.yaml`.
4. Generate a seed with that YAML.
5. Host the generated room.
6. Open the Shellipelago browser build and connect with the room host, port, slot, and password.
