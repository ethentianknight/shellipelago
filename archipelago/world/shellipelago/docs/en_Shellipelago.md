# Shellipelago

Shellipelago is a browser game where movement, major abilities, resources, presentation, and room access come from Archipelago checks.

The Archipelago world uses `src/data/map.json` for location names, requirements, room-ring logic, shops, enemies, and destructible checks.

Important progressive items include:

- Graphics x3
- Progressive Room x5
- Bombs x3
- Gun x2
- Sword x3
- Max HP and Max Rounds upgrades

Each Progressive Room opens the next square ring of rooms, up to ring 5.

Goals:

- Complete Final Run
- Build Tank
